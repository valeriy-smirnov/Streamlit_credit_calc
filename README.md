# Кредитный калькулятор (Streamlit)
Python-dev course 2025. Создание приложений на Streamlit

## О проекте
Проект представляет собой разработку приложения "кредитный калькулятор" с помощью streamlit

### Основные задачи:

Реализовать собственный кредитный калькулятор.
Напишите приложение на Python с использованием streamlit, которое будет обладать следующим функционалом:

считывание суммы кредита, процентной ставки, срока кредита и типа платежа по кредиту (аннуитетный и дифференциальный);
вычисление ежемесячного платежа по кредиту (для аннуитетного типа);
формирование графика платежей по кредиту в виде таблицы со следующими столбцами:
остаток долга,
ежемесячный платеж,
процентная часть,
долговая часть,
остаток долга на конец периода;
Дополнительно:
считывание даты первого платежа и добавление в таблицу даты каждого из платежей;
обработку случаев неправильного ввода;
использование механизмов условного рендеринга.
